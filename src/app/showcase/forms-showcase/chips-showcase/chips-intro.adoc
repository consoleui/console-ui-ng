= Chips 碎片输入

碎片输入，数组类型的的属性输入，如关键词。可以是对象或简单类型（一般是字符串），可以结合延迟加载和下拉选项实现更复杂的效果。

== 导入

[source, js]
----
import { CuiFormsModule } from 'consoleui';
----

[source, html]
